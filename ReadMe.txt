Code Description
This is a responsive navigation with a side bar and hamburger menu for tablet and mobile views. This project was created with JavaScript ES6 to code eventlisteners and resizing view window functions, along with a basic hamburger navigation click function to animate the sidebar into utilzing GSAP3 (Version: 3.10.2).

Notes:
- Basic general CSS styling for elements and added responsiveness for tablet/mobile devices.
- 2 Versions of hamburger icon and close icon in both black and white.
- GSAP3 animations were used to help with create better visual effects and animation purposes of showing Side Bar.

Technology: 
    - HTML5
    - CSS3
    - Vanilla JavaScript ES6+
    - GSAP3 (https://cdnjs.cloudflare.com/ajax/libs/gsap/3.10.2/gsap.min.js)
